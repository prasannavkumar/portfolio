Applied Projects

  • Reinforcement Learning Agent for Sonic The Hedgehog

    Developed an AI agent capable of playing Sonic The Hedgehog by leveraging reinforcement learning techniques. The agent learns how to play through trial and error, receiving rewards based on its success on going right. This project involved designing and training the decision-making process to achieve the desired outcome.

    Languages: Python

    Done as part of AI course at University of Waterloo

  • Binary Lifting Enhancement with Ghidra
    Enhanced an existing binary analysis tool by adding support for 64-bit systems using LLVM Intermediate Representation (IR). This involved extending functionalities within Ghidra through a combination of Python scripting and C++ development. The project improved compatibility and analysis precision for modern architectures, enabling more effective reverse engineering and binary lifting workflows.

    Technical Stack: Ghidra, LLVM IR, Python, C++

    Domain: Binary Analysis, Reverse Engineering

    Done as part of RA at University of Waterloo

Research and Publications

  • SecureCrypto: Framework for Preventing Cryptographic API Misuse in Java
    As part of my coursework during Master's, I developed a privacy-preserving machine learning protocol that enables secure and robust evaluation of decision trees while protecting both the clients input and the models structure with 2 classmates. Our protocol is designed for a multi-server environment and guarantees information-theoretic security of clients' input vectors through secret sharing.
    Cryptography: Secret Sharing, Reed-Solomon Error Correcting Codes
    Application Domain: Privacy-Preserving Machine Learning, Multi-Server Protocols
  • SDN Security: Measures, Analysis, and Comparison
    This paper examines the security landscape of SDN, analyzing how traditional network security issues translate to SDN environments. It highlights common vulnerabilities and discusses mitigation strategies, emphasizing that while SDN offers architectural advantages, security remains a critical and complex concern. The study aims to raise awareness and foster deeper consideration of SDN-specific security risks.
    Survey paper exploring security challenges in Software Defined Networking (SDN) done as part of Advanced Network Technology Course at University of Waterloo
    Application Domain: Network Security
